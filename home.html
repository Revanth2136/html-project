HTML interview questions

1.What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It defines elements like headings, paragraphs, links, and images to display web pages in browser.

2.What is the difference between HTML and XHTML?

HTML is flexible with syntax, while XHTML is stricter and follows XML rules. XHTML requires well-formed code (like closed tags and lowercase elements), making it more rigid than HTML.

4.What is the difference between id and class in HTML?

The id attribute uniquely identifies a single HTML element and is used once per page. The class attribute can be shared by multiple elements to apply common styles or behavior.

5.What are HTML input types?

HTML input types specify the kind of data an element accepts, like text, numbers, or dates. Examples include text, password, email, checkbox, radio, date, and submit.
